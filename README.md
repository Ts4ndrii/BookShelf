# 📚 BookShelf — Книги, що змінили історію
[![CI tests](https://github.com/Ts4ndrii/BookShelf/actions/workflows/ci-cd.yml/badge.svg)](https://github.com/Ts4ndrii/BookShelf/actions/workflows/ci-cd.yml)

BookShelf — це вебсайт-візитка та цифрова бібліотека, створена для презентації найвпливовіших літературних творів світу, а також для демонстрації власної книжкової колекції автора.

Проєкт слугує інформаційним хабом, де зібрані відомості про книги, які вплинули на хід історії. Окрім світової класики, тут представлена особиста бібліотека розробника, де особлива увага приділяється літературі про людську особистість, філософським роздумам та творам, що спонукають до самопізнання, на кшталт «Алхіміка». Також на сайті розміщено окрему сторінку з короткою біографічною довідкою про автора.

## 🛠 Стек технологій
Проєкт побудований на класичному вебстеку без використання важких фреймворків, що забезпечує максимальну швидкість та простоту:

- HTML5 — семантична структура сторінок.
- CSS3 — стилізація та адаптивний дизайн.
- Vanilla JavaScript — додавання базової інтерактивності (переходи на додаткову інформацію про книги).
- Git / GitHub — контроль версій та організація робочого простору.

## 🚀 Як переглянути (Deployment)
Оскільки це статичний вебсайт, він не потребує складного локального сервера чи збірки модулів.

1. Зклонуйте репозиторій на свій комп'ютер:
```
git clone https://github.com/Ts4ndrii/BookShelf.git
```
2. Перейдіть у папку проєкту:
```
cd BookShelf
```
3. Просто відкрийте файл index.html у будь-якому сучасному веббраузері (Chrome, Firefox, Safari).
